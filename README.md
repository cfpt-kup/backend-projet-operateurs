### Add files to commit
```
git add .
```

### Commit locally
```
git commit -m "MY MESSAGE !"
```

### Push to Github
```
git push -u origin main
```

### Running Docker Compose
```
docker-compose up --build
```

### Build Docker Image 
```
docker-compose build
```

### Restart the containers
```
docker-compose up -d
```

### Visualize README.md in VS Code
```
Cmd + Shift + V (Mac)

Ctrl + Shfit + V (Windows)
```
